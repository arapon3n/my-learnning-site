# 🌟 プログラミング学習サイトへようこそ！

このサイトでは、プログラミング初心者向けに **Unix コマンド** と **Git** の基礎を学べます。  
学習を進めることで、ターミナル操作やバージョン管理の基本を理解し、実践できるようになります。

## 🚀 学習内容
🔹 **Unix コマンド**  
- ファイル操作 (`ls`, `cd`, `cp`, `mv`, `rm`)
- 権限管理 (`chmod`, `chown`)
- プロセス管理 (`ps`, `kill`)
- その他の便利なコマンド (`grep`, `awk`, `sed`)

🔹 **Git**
- バージョン管理の基本 (`git init`, `git add`, `git commit`)
- リモートリポジトリとの連携 (`git push`, `git pull`)
- ブランチ管理 (`git branch`, `git checkout`, `git merge`)
- コンフリクト解決とワークフロー

### 🎯 **学習の進め方**
1. 各セクションの **概要** を読んで、基本概念を理解する
2. コマンドを **実際に試してみる**（ターミナル推奨）
3. 章ごとの **演習問題** にチャレンジする
4. 不明点があれば、復習して理解を深める

📌 **おすすめの環境**
- macOS / Linux のターミナル
- Windows の場合は WSL（Windows Subsystem for Linux）または Git Bash
