より高度な Unix コマンドを学びます。

## 🔍 検索・フィルタリング
| コマンド | 説明 |
|----------|----------------|
| `grep` | テキストを検索 |
| `find` | ファイルを検索 |
| `awk` | テキストの加工・処理 |
| `sed` | 文字列の置換 |

✅ **練習:**  
```sh
ps aux | grep "node"
find / -name "*.log"
```
